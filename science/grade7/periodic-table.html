The Element Master Chart! 🗺️

  The **Periodic Table** is a chart that organizes all the known **elements**, the basic building blocks of matter. It's organized so that elements with similar properties are placed together. Think of it like a library for atoms!

  Each element is made of only one type of atom. They are arranged by their **Atomic Number** (the number of protons) and by how their electrons are set up.

Reading an Element Tile 🤓

Every box on the table is called an element tile and tells you four important things. Here is a simple example for **Hydrogen** (H):

 
1
 
H
 
Hydrogen
 
1.008
     
  • **1. Atomic Number (1):** The number of **protons**. This is the element's ID!
    •  
  • **2. Element Symbol (H):** A one or two-letter short form.
    •  
  • **3. Element Name (Hydrogen):** The full name of the element.
    •  
  • **4. Atomic Mass (1.008):** The average mass of the atom (protons + neutrons).
