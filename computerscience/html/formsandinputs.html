📝 HTML Forms & Inputs Guide

Forms are how you collect user input in websites — login forms, contact forms, search bars, etc. Let’s see how to build them.

🔤 Basic Form Inputs

💡 Use <label> to associate text with inputs.
Common input types: text, password, email, number, submit

✅ Checkboxes & 🔘 Radio Buttons

✅ Checkboxes allow multiple selections.
🔘 Radio buttons allow only one selection within the same name.

📋 Dropdown & Textarea

🎯 <select> is for dropdowns.
📝 <textarea> is for multi-line text input.

🚀 Form Attributes

  • action → URL where form data is sent.
  • method → HTTP method (GET or POST).
  • required → Field must be filled before submit.
  • placeholder → Hint text inside input.
  • value → Default input value.
