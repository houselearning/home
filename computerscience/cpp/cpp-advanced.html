⚙️ C++ Advanced Concepts

Here’s where modern, high-performance C++ happens — covering smart pointers, lambdas, move semantics, constexpr, and more.

🧠 Smart Pointers (from <memory>)

💡 Use unique_ptr for sole ownership. shared_ptr for shared ownership. Avoid raw new/delete.

⚡ Lambda Functions

📌 Inline anonymous functions — pass them to STL algorithms or callbacks.

📦 Move Semantics & std::move

💡 Move avoids deep copies on large objects.

⏩ constexpr

  • constexpr int square(int x) { return x * x; }
  • Evaluated at compile-time if possible

📌 decltype & auto

  • auto n = 5; → compiler infers int
  • decltype(n) m = 10; → same type as n

📦 Namespaces & Anonymous Namespaces

  • Organize code into modules
  • Anonymous namespaces give internal linkage (like static globals)

🎛️ RAII (Resource Acquisition Is Initialization)

  • Use object lifetime to manage resource cleanup
  • Smart pointers + destructors automatically free memory when out of scope
💡 Modern C++ prefers no raw new/delete. Use unique_ptr, shared_ptr, and RAII patterns.

🧑‍🔬 Bonus: Type Traits & Template Metaprogramming

  • std::is_integral<T>::value
  • std::enable_if<...>
  • Conditionally enable functions/types at compile time
