🔧 Common Array Methods Cheat Sheet

  • push() — Adds an element to the end
  • pop() — Removes the last element
  • shift() — Removes the first element
  • unshift() — Adds an element at the start
  • slice(start, end) — Returns a section of the array
  • splice(start, deleteCount) — Adds/Removes elements
  • forEach(callback) — Loops through each element
  • map(callback) — Creates a new array with modified items

⚡ Quick Tips

  • Arrays can hold any data type — numbers, strings, objects, even other arrays.
  • Use length property to get array size: array.length
  • Arrays are dynamic in JS — no need to declare a fixed size.
  • Immutable methods like map() and filter() return new arrays.
