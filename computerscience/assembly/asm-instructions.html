📖 Assembly Instructions & Directives

Assembly language is made up of two main things: instructions that tell the CPU what to do, and directives that guide the assembler on how to process your code.

⚙️ Common Instructions

  • mov destination, source — Moves data from one place to another
  • add destination, source — Adds a value to a register
  • sub destination, source — Subtracts a value
  • int 0x80 — Performs a Linux system call
  • jmp label — Jumps to a label in code

📌 Common Directives

  • section .data — Holds initialized data
  • section .bss — Holds uninitialized data
  • section .text — Contains the code/instructions
  • global _start — Marks the program's entry point
  • db, dw, dd — Define byte, word, doubleword
