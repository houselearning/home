⚙️ Lesson: C# Functions & Methods

Functions (or methods) in C# are blocks of reusable code that perform specific tasks. They help organize your code, avoid repetition, and make programs easier to read and maintain.

Function Syntax

returnType FunctionName(parameters) {
    // function body
    return value;
}

Explanation:

  • returnType — data type returned by the function (use void if no value is returned)
  • FunctionName — your function's name
  • parameters — input variables (optional)
  • return — value returned from the function (if any)

Example: Adding Two Numbers

int AddNumbers(int a, int b) {
    return a + b;
}

This function takes two integers and returns their sum.
