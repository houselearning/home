Lesson: C# Object-Oriented Programming (OOP)

Object-Oriented Programming (OOP) is a programming paradigm based on the concept of "objects," which can contain both data and methods. The four main principles of OOP are:

  • Encapsulation: Bundling the data (attributes) and methods (functions) that operate on the data into a single unit called a class.
  • Abstraction: Hiding the complexity of the system and exposing only the necessary details.
  • Inheritance: The ability to create a new class from an existing class, inheriting its properties and behaviors.
  • Polymorphism: The ability to use a derived class object in the place of a base class object, enabling one interface to be used for different types.

Class and Object

A class is a blueprint for creating objects, and an object is an instance of a class.

Here's how a class is defined in C#: 

class Car {
    public string make;
    public string model;

    public void Drive() {
        Console.WriteLine("The car is driving.");
    }
}

In the above example, Car is a class with two attributes (make, model) and one method (Drive).

Creating Objects

To create an object, we use the new keyword followed by the class name: 

Car myCar = new Car();
myCar.make = "Toyota";
myCar.model = "Corolla";
myCar.Drive();

This code creates an instance of the Car class and assigns values to its properties, then calls its method to simulate the car driving.
