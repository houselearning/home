Groups of Equal Numbers

Multiplication is just a fast way to add the same number over and over again!


The Easy Rules

The Zero Rule: Any number times 0 is ALWAYS 0.
Example: 3 x 0 = 0 (Three groups of nothing)
The One Rule: Any number times 1 stays the SAME.
Example: 3 x 1 = 3 (Three groups of one)
The Two Rule: Multiplication by 2 is just DOUBLING.
Example: 3 x 2 = 6 (3 + 3)
The Three Rule: Think of it as tripling or 3 groups.
Example: 3 x 3 = 9 (3 + 3 + 3)
